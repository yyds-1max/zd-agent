# 样例数据说明

本目录包含用于 Demo 的模拟知识数据，当前已满足：
- 12 份样例知识（范围要求 8-15）
- 覆盖四类来源：制度（policy）、项目（project）、FAQ（faq）、聊天结论（chat_summary）
- 至少 2 组同主题不同版本文档（用于版本感知演示）
- 至少 3 种角色（employee、finance、pm）用于权限差异演示



## 版本感知样本
1. 差旅报销管理制度：`policy-travel-v1` -> `policy-travel-v2`
2. 采购审批制度：`policy-procurement-v1` -> `policy-procurement-v2`

## 聊天结论样本
聊天类数据均为“最终结论”格式，已去除原始长对话，示例：
- `chat-a-delivery-final-202604`
- `chat-finance-close-final-202604`
